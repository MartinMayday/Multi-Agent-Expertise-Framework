# Skills

Specialized capabilities that agents can leverage.

## Structure

Each skill is a reusable capability with:
- Clear interface (what it does)
- Requirements (what it needs)
- Tools it uses
- Examples of usage

## Available Skills

### research-specialist/

Research and information gathering.

**Tools**: Web search, content extraction, BM25 search
**Capabilities**: Literature review, fact-finding, trend analysis

### code-analyst/

Code understanding and analysis.

**Tools**: Git, AST parsing, dependency analysis
**Capabilities**: Code review, technical debt analysis, refactoring identification

### devops-engineer/

Infrastructure and deployment expertise.

**Tools**: Docker, Kubernetes, monitoring APIs
**Capabilities**: CI/CD setup, scalability analysis, observability design
