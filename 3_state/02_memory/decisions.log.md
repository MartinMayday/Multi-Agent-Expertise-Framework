# Decision Log

Architectural Decision Records (ADRs) for this project.

## Format

Each decision includes:
- **Date**: When decided
- **Decision**: What was decided
- **Context**: Why (alternatives considered)
- **Consequences**: Positive and negative impacts
- **Status**: Accepted, Deprecated, Superseded

---

## ADR Template

### ADR-[number]: [Decision Title]

**Date**: YYYY-MM-DD

**Decision**:
[What was decided]

**Context**:
[Why this decision was needed. What alternatives were considered?]

**Consequences**:
- Positive: [benefits]
- Negative: [tradeoffs]

**Status**: Accepted
